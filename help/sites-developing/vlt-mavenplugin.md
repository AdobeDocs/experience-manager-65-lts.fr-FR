# Plug-in Adobe Content Package Maven {#adobe-content-package-maven-plugin}

Le plug-in de module de contenu Maven d’Adobe permet d’intégrer les tâches de déploiement et de gestion de modules dans vos projets Maven.

Le plug-in Adobe Content Package Maven déploie les packages construits sur AEM et automatise les tâches que vous effectueriez normalement dans le gestionnaire de packages AEM.
