# Fragments de code {#snippets}

## Préparation en cours {#work-in-progress}

La documentation d’Adobe Experience Manager 6.6 est en cours de préparation.AEM 6.6 n’est pas en disponibilité générale et cette documentation n’est pas encore publique.

## L’éditeur universel est préférable à l’éditeur de SPA. {#ue-over-spa}

>[!IMPORTANT]
>
>L’éditeur de SPA a été abandonné pour les nouveaux projets.Il est toujours pris en charge par Adobe pour les projets existants, mais ne doit pas être utilisé pour de nouveaux projets. Les éditeurs privilégiés pour la gestion du contenu découplé dans AEM sont désormais les suivants :
>
>* L’[éditeur universel](/help/sites-developing/universal-editor/introduction.md) pour modifier visuellement du contenu découplé.
>* L’[éditeur de fragment de contenu](/help/assets/content-fragments/content-fragments-managing.md) pour la modification du contenu découplé à partir des formulaires.
